3.1.  Stats

3.1.1. Internal Stat Values

The Stats service collects statistical information from various points inside Sorento. This allows a website author to quickly pull fx. online users or total page views directly from a template.

Stat values are updated every 3 minutes automatically by the Stat service. It is also possible for a single part of Sorento to force an update if something happens. Eg. if a User is created, "SorentoLib.User.TotalUsers" value will be updated immediately.
